# Joseph Chukwunazam Kelvin - Portfolio Website

A modern, responsive personal portfolio website showcasing the work and skills of Joseph Chukwunazam Kelvin - a Frontend Developer, Graphic Designer, and Tech Enthusiast.

## 🎨 Features

- **Modern Design**: Dark modern UI with vibrant blue/purple gradient accents
- **Fully Responsive**: Optimized for mobile, tablet, and desktop devices
- **Smooth Animations**: Framer Motion animations for engaging user experience
- **Glassmorphism UI**: Modern glassmorphism effects with soft shadows
- **Dark Mode Aesthetic**: Professional dark theme with gradient accents
- **SEO Friendly**: Semantic HTML and optimized structure
- **Fast Performance**: Optimized build with Vite

## 🛠️ Tech Stack

- **React 18**: Modern JavaScript library for UI
- **Vite**: Fast build tool for modern web development
- **Tailwind CSS**: Utility-first CSS framework
- **Framer Motion**: Smooth animations library
- **Lucide React**: Beautiful icon library

## 🚀 Getting Started

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
cd "g:\My Drive\Professional Foundation\MY WEBSITE"
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open your browser and navigate to `http://localhost:3000`

## 📦 Build

To create an optimized production build:

```bash
npm run build
```

To preview the production build:

```bash
npm run preview
```

## 🎨 Customization

### Colors
Modify the color scheme in `tailwind.config.js`:
- Primary colors: `from-blue-500 to-blue-600`
- Accent colors: `from-purple-500 to-purple-600`
- Dark background: `dark-900`, `dark-800`

### Fonts
The project uses:
- **Poppins** - Primary font family
- **Inter** - Secondary font family

### Content
Update the content in individual component files:
- Hero content: `src/components/sections/Hero.jsx`
- Projects: `src/components/sections/Projects.jsx`
- Contact info: `src/components/sections/Contact.jsx`
